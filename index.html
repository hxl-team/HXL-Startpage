The HXL documentation contains introductory tutorials for producing HXL and using HXL in your own applications, including code snippets to learn by example. It also includes a crash course on Linked Data and references for further reading.

The HXL situation and response standard is an RDF vocabulary used for the semantic annotation of HXL data. The standard defines the different classes and properties available in HXL.

Updates about the project are posted and discussed on the Google group. The code for the tools around HXL is available on GitHub. This is also the right place to report issues with any of the HXL tools.
